SyncSMDemo application
======================

This demo is intended to be used to show synchronisation using the sync manager events (SM2/SM3).

Features:
---------
- only works if the link layer operates in interrupt mode.
- toogles the outputs in every cycle on the first Beckhoff EL2002, EL2004, EL2008 device.
- default cycle time: 1 ms 

The bus timimg is described in details in the chapter "Cyclic and acyclic frames – Link layer in interrupt mode"
in the user manual.
